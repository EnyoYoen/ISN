# Projet Final ISN
Projet ISN 2A

## Dépendances :

Ce projet dépend de Pygame et de quelques autres dépendances. Vous pouvez l'installer en utilisant pip avec la commande suivante :
```bash
python -m pip install -r requirements.txt
```

## Lancement du jeu :
Dans la racine du dossier :
```bash
python src/main.py
```
